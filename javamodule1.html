1. Introduction to Java

Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995. It is known for its portability, performance, and robustness. Java applications can be run on any device that has a Java Virtual Machine (JVM) installed, making it a popular choice for developing cross-platform applications.

Key features of Java include:

  • Object-Oriented: Everything in Java is an object, which helps in code reusability and modularity.
  • Platform Independence: Java code can run on any platform without the need for recompilation.
  • Automatic Memory Management: Java has built-in garbage collection that helps manage memory efficiently.
  • Rich API: Java provides a vast array of libraries and frameworks for various applications.

Example - Basic Java Program:


public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

This simple program prints "Hello, World!" to the console.

2. Variables and Data Types

Variables in Java are used to store data values. Each variable has a data type that defines the kind of data it can hold. Common data types in Java include:

  • int: For storing integer values (e.g., 10, -5).
  • double: For storing decimal values (e.g., 10.5, -3.14).
  • char: For storing a single character (e.g., 'A', 'b').
  • boolean: For storing true or false values.
  • String: For storing a sequence of characters (e.g., "Hello").

Example - Declaring Variables:


public class VariableExample {
    public static void main(String[] args) {
        int age = 25;
        double salary = 50000.50;
        char grade = 'A';
        boolean isJavaFun = true;
        String name = "John";
        
        System.out.println("Name: " + name + ", Age: " + age);
    }
}

This example demonstrates how to declare and initialize different types of variables in Java.

3. Operators in Java

Operators are special symbols that perform operations on variables and values. Java supports several types of operators:

  • Arithmetic Operators: Used to perform basic mathematical operations like addition (+), subtraction (-), multiplication (*), and division (/).
  • Relational Operators: Used to compare two values (e.g., ==, !=, >, <).
  • Logical Operators: Used to combine multiple boolean expressions (e.g., &&, ||, !).
  • Assignment Operators: Used to assign values to variables (e.g., =, +=, -=).

Example - Using Operators:


public class OperatorExample {
    public static void main(String[] args) {
        int a = 10;
        int b = 5;
        
        int sum = a + b; // Addition
        boolean isGreater = a > b; // Relational
        boolean isTrue = (a > b) && (b < 10); // Logical
        
        System.out.println("Sum: " + sum);
        System.out.println("Is A greater than B? " + isGreater);
        System.out.println("Is A greater than B and B less than 10? " + isTrue);
    }
}

This example shows how to use various operators in Java.

4. Control Statements

Control statements allow you to dictate the flow of execution of a Java program. The primary control statements include:

  • If Statement: Executes a block of code if its condition is true.
  • Switch Statement: Executes a block of code based on the value of a variable.
  • For Loop: Repeats a block of code a specified number of times.
  • While Loop: Repeats a block of code while its condition is true.

Example - Using Control Statements:


public class ControlStatementExample {
    public static void main(String[] args) {
        int number = 10;
        
        // If Statement
        if (number > 0) {
            System.out.println("Number is positive.");
        }
        
        // Switch Statement
        switch (number) {
            case 1:
                System.out.println("Number is one.");
                break;
            case 10:
                System.out.println("Number is ten.");
                break;
            default:
                System.out.println("Number is neither one nor ten.");
        }
        
        // For Loop
        System.out.println("Numbers from 1 to 5:");
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}

This example illustrates how to use different control statements in Java.
