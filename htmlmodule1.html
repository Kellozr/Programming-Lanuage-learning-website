1. What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure of web pages using elements, which include tags like <h1>, <p>, and <div>. HTML elements are represented by tags that start with < and end with >.

HTML is the backbone of the web, allowing us to create links, embed images, structure content, and much more. A basic understanding of HTML is essential for web development.

2. Structure of an HTML Document

An HTML document is structured using various tags and elements. Here is a typical structure:


<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to My Website</h1>
        <p>This is a paragraph.</p>
    </body>
</html>

The structure includes the <html> tag that wraps everything, the <head> section containing metadata, and the <body> section, where the visible content resides.

3. Common HTML Tags

Here are some of the most commonly used HTML tags:

  • <h1> to <h6>: Heading tags, <h1> being the largest and <h6> the smallest.
  • <p>: Defines a paragraph.
  • <a>: Creates hyperlinks.
  • <img>: Embeds images.
  • <div>: Defines a division or section.

Example:


<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<a href="https://www.example.com">Visit Example</a>

4. Attributes in HTML

Attributes provide additional information about HTML elements. They are always included in the opening tag and have a name and value. Common attributes include href for links, src for images, and alt for alternative text.

Example:


<a href="https://www.example.com">Click here</a>
<img src="image.jpg" alt="A description of the image">

5. HTML Comments

Comments in HTML are useful for adding notes to your code. They are not displayed in the browser and help in making the code more readable. HTML comments are written using <!-- -->.

Example:


<!-- This is a comment -->
<p>This is a visible paragraph.</p>

6. Creating Your First HTML Page

Let's create a simple HTML page that includes a heading, paragraph, image, and a link. Save this code in a file with the extension .html and open it in a web browser to see the result. 


<!DOCTYPE html>
<html>
    <head>
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Hello World!</h1>
        <p>Welcome to learning HTML.</p>
        <img src="hello-world.jpg" alt="Hello World Image">
        <a href="https://www.example.com">Learn more</a>
    </body>
</html>
