1. Introduction to Input and Output

Input and Output (I/O) in Java refers to the process of receiving data from a user or a file (input) and displaying data on the screen or saving it to a file (output). Java provides several classes and methods for handling I/O operations.

The most common classes for I/O operations are:

  • Scanner: Used for reading input from various sources like keyboard, files, etc.
  • PrintStream: Used for writing output to various destinations.

2. Using the Scanner Class

The Scanner class is part of the java.util package and is used to read input from various sources. Here is an example of using the Scanner class to read user input: 


import java.util.Scanner;

public class InputExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your name: ");
        String name = scanner.nextLine();
        System.out.println("Hello, " + name + "!");
        scanner.close();
    }
}

4. Reading Data from Files

Java provides classes such as FileReader and BufferedReader for reading data from files. Here's a simple example: 


import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class FileReadExample {
    public static void main(String[] args) {
        try (BufferedReader reader = new BufferedReader(new FileReader("input.txt"))) {
            String line;
            while ((line = reader.readLine()) != null) {
                System.out.println(line);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

5. Writing Data to Files

To write data to files, you can use the FileWriter and BufferedWriter classes. Here's how you can write text to a file: 


import java.io.BufferedWriter;
import java.io.FileWriter;
import java.io.IOException;

public class FileWriteExample {
    public static void main(String[] args) {
        try (BufferedWriter writer = new BufferedWriter(new FileWriter("output.txt"))) {
            writer.write("Hello, World!");
            writer.newLine();
            writer.write("Welcome to file handling in Java.");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
