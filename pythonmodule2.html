1. Control Structures

Control structures are essential components in programming that dictate the flow of execution of the code. In Python, there are primarily three types of control structures: conditional statements, loops, and control flow statements.

1.1 Conditional Statements

Conditional statements allow you to execute certain pieces of code based on specific conditions. The most commonly used conditional statement in Python is the if statement.

if condition:
    # execute this block of code
elif another_condition:
    # execute this block of code
else:
    # execute this block of code

Example:

age = 18
if age < 18:
    print("You are a minor.")
elif age == 18:
    print("You are just an adult.")
else:
    print("You are an adult.")

1.2 Loops

Loops allow you to execute a block of code multiple times. Python provides two types of loops: for loops and while loops.

For Loop

The for loop is used to iterate over a sequence (like a list or string).

for item in sequence:
    # execute this block of code

Example:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

While Loop

The while loop continues to execute as long as a specified condition is True.

while condition:
    # execute this block of code

Example:

count = 0
while count < 5:
    print(count)
    count += 1

1.3 Control Flow Statements

Control flow statements alter the flow of execution in loops. Python provides several control flow statements such as break and continue.

Break Statement

The break statement is used to exit a loop prematurely.

for num in range(10):
    if num == 5:
        break
    print(num)

Continue Statement

The continue statement skips the current iteration and continues with the next iteration of the loop.

for num in range(10):
    if num % 2 == 0:
        continue
    print(num)

2. Functions

Functions are reusable blocks of code that perform a specific task. They help in organizing code and improving readability.

2.1 Defining Functions

You can define a function using the def keyword, followed by the function name and parentheses.

def function_name(parameters):
    # code block
    return result

Example:

def greet(name):
    return f"Hello, {name}!"

message = greet("Alice")
print(message)

2.2 Function Parameters

Functions can accept parameters to pass data. You can define default parameters, variable-length arguments, and keyword arguments:

Default Parameters

def greet(name="World"):
    return f"Hello, {name}!"

Variable-Length Arguments

def add(*numbers):
    return sum(numbers)

Keyword Arguments

def describe_person(name, age):
    return f"{name} is {age} years old."
