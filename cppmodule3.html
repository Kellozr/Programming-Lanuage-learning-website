1. Introduction to Control Statements

Control statements in C++ allow us to alter the flow of execution of our programs based on certain conditions or repetitions. These statements help in making decisions, looping through a block of code multiple times, or jumping to a specific part of the code.

2. Conditional Statements

Conditional statements help to make decisions in a program based on specific conditions. The most commonly used conditional statements are:

  • if Statement: Executes a block of code if the condition is true. 
    
int age = 18;
if (age >= 18) {
    cout << "You are eligible to vote.";
}
  • if-else Statement: Executes one block of code if the condition is true, otherwise executes another block. 
    
int num = 5;
if (num % 2 == 0) {
    cout << "Even number.";
} else {
    cout << "Odd number.";
}
  • else if Ladder: Used for multiple conditions. 
    
int score = 75;
if (score >= 90) {
    cout << "A grade";
} else if (score >= 75) {
    cout << "B grade";
} else {
    cout << "C grade";
}
  • switch Statement: Allows multi-way branching based on the value of a variable. 
    
int day = 3;
switch (day) {
    case 1: cout << "Monday"; break;
    case 2: cout << "Tuesday"; break;
    case 3: cout << "Wednesday"; break;
    default: cout << "Invalid day";
}

3. Looping Statements

Looping statements in C++ allow you to execute a block of code repeatedly as long as a certain condition is met. The main types of loops are:

  • for Loop: Best used when the number of iterations is known. 
    
for (int i = 1; i <= 5; i++) {
    cout << i << " ";
}
  • while Loop: Repeats a block of code as long as the condition is true. 
    
int i = 1;
while (i <= 5) {
    cout << i << " ";
    i++;
}
  • do-while Loop: Similar to a while loop, but ensures the code runs at least once. 
    
int i = 1;
do {
    cout << i << " ";
    i++;
} while (i <= 5);

4. Jump Statements

Jump statements in C++ help to change the normal flow of execution in loops or functions. Commonly used jump statements are:

  • break: Exits the loop or switch statement. 
    
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break;
    }
    cout << i << " ";
}
  • continue: Skips the current iteration of the loop and continues with the next. 
    
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue;
    }
    cout << i << " ";
}
  • return: Exits a function and optionally returns a value. 
    
int add(int a, int b) {
    return a + b;
}

5. Example: Control Statements in Action

Here is a simple program that demonstrates the use of if-else and loops:


#include <iostream>
using namespace std;

int main() {
    int number;
    cout << "Enter a positive number: ";
    cin >> number;

    if (number > 0) {
        cout << "Counting from 1 to " << number << ": " << endl;
        for (int i = 1; i <= number; i++) {
            cout << i << " ";
        }
    } else {
        cout << "Please enter a positive number." << endl;
    }
    return 0;
}

This program prompts the user to enter a positive number and counts from 1 to the entered number using an if-else condition and a for loop.

6. Conclusion

Control statements are fundamental to creating dynamic and efficient programs in C++. They allow you to control the flow of execution based on conditions and repetitions. Practice using different control statements to master their application in real-world scenarios.
