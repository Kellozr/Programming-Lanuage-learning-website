1. Understanding Data Types

In C++, data types are essential for defining the type of data a variable can hold. The primary data types in C++ can be classified into three categories:

  • Primitive Data Types:
    • int: Used for integers. Example: int age = 25;
    • float: Used for floating-point numbers. Example: float salary = 50000.50;
    • double: Used for double-precision floating-point numbers. Example: double pi = 3.14159;
    • char: Used for single characters. Example: char grade = 'A';
    • bool: Used for boolean values. Example: bool isPassed = true;
  • Derived Data Types:
    • array: A collection of elements of the same type. Example: int numbers[5] = {1, 2, 3, 4, 5};
    • pointer: A variable that holds the address of another variable. Example: int *ptr = &age;
    • structure: A user-defined data type that groups different data types. Example: 
      
struct Person {
    string name;
    int age;
    float height;
};
  • User-Defined Data Types:
    • class: Defines a blueprint for objects. Example: 
      
class Car {
public:
    string brand;
    int year;
};
    • enum: A user-defined type consisting of a set of named integer constants. Example: 
      
enum Day {Sun, Mon, Tue, Wed, Thu, Fri, Sat};

Choosing the correct data type is crucial for memory management and performance optimization.

2. Declaring Variables

A variable in C++ is a named storage location in memory that can hold data. To declare a variable, you need to specify its data type and name. The syntax is: 

data_type variable_name;

For example:

int age;

You can also initialize a variable at declaration:

float weight = 65.5;

Multiple variables can be declared in one line:

int a, b, c;

Variable names must start with a letter or underscore, followed by letters, digits, or underscores. C++ is case-sensitive, so Age and age are different.

3. Operators in C++

C++ provides a rich set of operators used to perform operations on variables and values:

  • Arithmetic Operators:
    • +: Addition
    • -: Subtraction
    • *: Multiplication
    • /: Division
    • %: Modulus
  • Relational Operators:
    • ==: Equal to
    • !=: Not equal to
    • >: Greater than
    • <: Less than
    • >=: Greater than or equal to
    • <=: Less than or equal to
  • Logical Operators:
    • &&: Logical AND
    • ||: Logical OR
    • !: Logical NOT
