1. Structure of a Java Program

Understanding the structure of a Java program is essential for writing and organizing code effectively. A typical Java program consists of classes, methods, variables, and statements. Here is a breakdown of a simple Java program: 


public class Example {
    public static void main(String[] args) {
        System.out.println("Welcome to Java Programming!");
    }
}

Let's break down the components of this code:

  • Class Declaration: The program starts with public class Example. This defines a class named Example, and in Java, all code must be inside a class.
  • Main Method: The main method (public static void main(String[] args)) is the entry point of any Java application. When you run a Java program, the JVM looks for the main method to start execution.
  • Statements: Inside the main method, System.out.println prints text to the console. Each statement in Java ends with a semicolon (;).

2. Java Data Types

Java is a strongly-typed language, which means every variable must have a declared type. There are two categories of data types in Java:

  • Primitive Data Types: Java provides eight primitive data types, which represent simple values:
    • byte: 8-bit integer, range from -128 to 127.
    • short: 16-bit integer, range from -32,768 to 32,767.
    • int: 32-bit integer, range from -231 to 231-1.
    • long: 64-bit integer, range from -263 to 263-1.
    • float: 32-bit floating-point number, used for precise values like currency.
    • double: 64-bit floating-point number, commonly used for decimal values.
    • char: 16-bit Unicode character, used to store characters.
    • boolean: Represents true or false.
  • Non-Primitive Data Types: These are reference types and include classes, arrays, strings, and interfaces. They store references to the memory locations where the actual data is stored.

3. Variables in Java

Variables are used to store data in Java. Each variable must be declared with a type before use. Here are different types of variables:

  • Local Variables: Declared inside methods or blocks and are accessible only within those methods or blocks. They do not retain values after the method completes.
  • Instance Variables: Declared inside a class but outside methods. These variables are specific to each object of the class.
  • Static Variables: Declared with the static keyword and belong to the class rather than an instance. All instances of the class share the same value of a static variable.

Example of declaring variables:


int age = 25;  // Local variable
double salary = 45000.50;  // Local variable
static String companyName = "Tech Solutions";  // Static variable

4. Control Flow Statements

Control flow statements allow you to control the execution flow of a program. Java provides several control flow mechanisms:

  • If-Else Statement: Used to execute a block of code conditionally based on whether an expression evaluates to true or false. 
    
int number = 10;
if (number > 0) {
    System.out.println("Positive number");
} else {
    System.out.println("Negative number or zero");
}
  • Switch Statement: Allows multi-way branching. It is used when there are multiple possible outcomes for a given expression. 
    
int day = 3;
switch (day) {
    case 1:
        System.out.println("Monday");
        break;
    case 2:
        System.out.println("Tuesday");
        break;
    default:
        System.out.println("Other day");
}
  • For Loop: Used for iterating over a block of code a specific number of times. 
    
for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}
  • While Loop: Repeats a block of code as long as a condition is true. 
    
int count = 0;
while (count < 5) {
    System.out.println("Count: " + count);
    count++;
}
  • Do-While Loop: Similar to the while loop, but ensures that the block of code is executed at least once. 
    
int number = 0;
do {
    System.out.println("Number: " + number);
    number++;
} while (number < 3);

5. Basic Operators in Java

Java provides a rich set of operators for performing arithmetic, comparison, and logical operations:

  • Arithmetic Operators: Used for mathematical operations. 
    
int a = 10, b = 5;
System.out.println(a + b);  // Addition
System.out.println(a - b);  // Subtraction
System.out.println(a * b);  // Multiplication
System.out.println(a / b);  // Division
  • Comparison Operators: Used to compare two values. 
    
System.out.println(a == b);  // Equal to
System.out.println(a != b);  // Not equal to
  • Logical Operators: Used for logical operations. 
    
boolean result = (a > b) && (b > 0);  // Logical AND
