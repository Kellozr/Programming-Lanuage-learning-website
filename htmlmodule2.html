1. Understanding HTML Elements

An HTML element consists of a start tag, content, and an end tag, like <p>Content</p>. Elements can be nested to form a hierarchy.

2. Block-level vs. Inline Elements

Elements are classified as:

  • Block-level: Takes full width, e.g., <div>, <p>.
  • Inline: Takes only necessary width, e.g., <span>, <a>.

<div>
    <p>Paragraph inside a block-level element.</p>
    <span>Inline element.</span>
</div>

3. HTML Attributes

Attributes provide additional info, written within the start tag. Examples include href and src.


<a href="https://www.example.com" target="_blank">Visit Example</a>
<img src="image.jpg" alt="Image description">

4. Adding Multimedia

Use <img> for images and <video> for videos:


<img src="nature.jpg" alt="Nature" width="300" height="200">
<video controls width="500">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

5. Using Lists

HTML supports:

  • Unordered Lists: <ul>
    • 
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
  • Ordered Lists: <ol>
    • 
<ol>
    <li>Step 1</li>
    <li>Step 2</li>
</ol>

6. HTML Tables

Use <table>, <tr>, and <td> for structured data:


<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>
